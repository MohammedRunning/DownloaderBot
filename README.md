# Telegram Media Downloader Bot 🤖

بوت تيليجرام سريع وخفيف مبني باستخدام Python و aiogram 3.x لتحميل الفيديوهات مباشرة من Instagram و TikTok وإرسالها للمستخدم داخل تيليجرام بأعلى جودة متاحة.

---

# ✨ المميزات

* تحميل فيديوهات Instagram Reels.
* تحميل فيديوهات Instagram Posts.
* تحميل فيديوهات TikTok.
* اكتشاف المنصة تلقائياً من الرابط.
* إرسال الفيديو مباشرة داخل تيليجرام.
* دعم أعلى جودة متاحة تلقائياً.
* رسائل حالة أثناء التنفيذ:

  * 📥 استخراج البيانات
  * ⬇️ تحميل الفيديو
  * 📤 إرسال الفيديو
* نظام Providers احتياطي لضمان استمرارية التحميل.
* حذف الملفات المؤقتة تلقائياً بعد الإرسال.
* معالجة أخطاء متقدمة وسجلات Logging.
* مبني بالكامل باستخدام Async لتحسين الأداء.

# 🛠 المتطلبات

* Python 3.12+
* Telegram Bot Token
* FFmpeg (اختياري لبعض عمليات المعالجة)
* اتصال إنترنت مستقر

# 📦 التثبيت

## تثبيت المكتبات

```bash
pip install -r requirements.txt
```

## إعداد ملف البيئة

أنشئ ملف:

```env
.env
```

وأضف:

```env
BOT_TOKEN=YOUR_BOT_TOKEN
```

# 🚀 التشغيل


2. أرسل رابط Instagram أو TikTok.
3. انتظر حتى اكتمال المعالجة.
4. استلم الفيديو مباشرة داخل تيليجرام.

# 📝 ملاحظات

* الحسابات الخاصة (Private Accounts) غير مدعومة.
* بعض المقاطع قد تكون غير متاحة بسبب قيود المنصة الأصلية.
* يوصى بتحديث yt-dlp بشكل دوري:

```bash
pip install -U yt-dlp
```
