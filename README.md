# بوت تحميل فيديوهات إنستقرام لتيليجرام 🤖📸

بوت تيليجرام بسيط وسريع جداً، مبني باستخدام لغة بايثون ومكتبة `aiogram 3.x`. يقوم البوت بتحميل الفيديوهات والـ Reels والمنشورات المرئية من إنستقرام بجودة عالية وإرسالها مباشرة داخل المحادثة كملف فيديو.

---

## ⚡ المميزات

* **سهولة الاستخدام:** بمجرد إرسال رابط المنشور يتم استخراج الفيديو وتحميله وإرساله مباشرة.
* **تحديثات الحالة الفورية:** يعرض للمستخدم مراحل العمل (📥 استخراج -> ⬇️ تحميل -> 📤 إرسال).
* **معالجة متزامنة غير حظرية (Async):** يعتمد على `asyncio` لتجنب بطء الاستجابة لطلبات المستخدمين.
* **دعم أعلى جودة:** يستعين بمكتبة `yt-dlp` القوية لتحديد وتحميل أفضل جودة.
* **إدارة المساحة تلقائياً:** يقوم بمسح الفيديوهات المحملة فوراً بعد الإرسال لعدم شغل مساحة السيرفر.
* **تضمين تفاصيل الفيديو:** يستخرج الطول والعرض والمدة لكي يظهر الفيديو بشكل منسق مع مشغل تيليجرام.

---

## 🛠️ المتطلبات الأساسية

* **Python 3.12+**
* رمز توكن البوت (Bot Token) من [BotFather](https://t.me/BotFather).

---

## 🚀 طريقة التثبيت والتشغيل

### 1. تثبيت الحزم المطلوبة
افتح الطرفية (Terminal) أو سطر الأوامر (CMD) في مجلد المشروع، ونفذ الأمر التالي لتثبيت المكتبات اللازمة:
```bash
pip install -r requirements.txt
```

### 2. إعداد المتغيرات البيئية
قم بتغيير اسم الملف `.env` (أو أنشئه إذا لم يكن موجوداً) وضع توكن البوت الخاص بك:
```env
BOT_TOKEN=ضع_توكن_البوت_الخاص_بك_هنا
```

### 3. تشغيل البوت
قم بتشغيل الملف الرئيسي `bot.py` لبدء البوت:
```bash
python bot.py
```

## 📝 ملاحظات هامة

1. **الروابط الخاصة (Private):** البوت يدعم تحميل الفيديوهات من الحسابات العامة فقط؛ الحسابات الخاصة والمقيدة تتطلب تسجيل الدخول ولا يمكن تحميلها مباشرة.
2. **حدود الحجم:** يلتزم البوت بحد أقصى لحجم الملف المرسل وهو **50 ميجابايت** كحد أقصى مفروض من قبل خوادم تيليجرام للبوتات العادية.
3. **تحديث `yt-dlp`:** يتغير موقع إنستقرام بشكل مستمر. في حال حدوث أي خطأ في استخراج الروابط مستقبلاً، تأكد من تحديث مكتبة `yt-dlp` عبر الأداة:
   ```bash
   pip install --upgrade yt-dlp
   ```
